nRF51822 EK TM

简介

nRF51822 EK TM是一个智能蓝牙开发板，您的手机可以通过蓝牙与它连接，起到远程控制、数据读取等目的。 作为新一代开发板，它易于上手，开发者不需要懂三极管、单片机、IOS、Android....，只要写一丁点儿网页就可以进行开发。什么？您说您网页也不会？没关系，网页编程比您想象的要简单，学习几个例子就会了。 本文将以扫描蓝牙设备、温度传感器、蜂鸣器、LED灯作为DEMO，带您进入nRF51822 EK TM的世界。以后，您可以发挥想象力，用它做更多的事情。

假设您现在手边还未获得开发板，我们提供了仿真模式来模拟nRF51822 EK TM的行为。启动仿真模式，请打开开关。

source code

var canvas = document.getElementById('picker'); var ctx = canvas.getContext('2d'); var image = new Image(); image.onload = function(){ ctx.drawImage(image, 0, 0, image.width, image.height); } var imageSrc = 'img/colorwheel1.png'; image.src = imageSrc; $('#picker').click(function(e) { var canvasOffset = $(canvas).offset(); var canvasX = Math.floor(e.pageX - canvasOffset.left); var canvasY = Math.floor(e.pageY - canvasOffset.top); var imageData = ctx.getImageData(canvasX, canvasY, 1, 1); var pixel = imageData.data; var pixelColor = "rgb("+pixel[0]+", "+pixel[1]+", "+pixel[2]+")"; var dColor = pixel[2] + 256 * pixel[1] + 65536 * pixel[0]; $('#hexVal').css('color',' #' + ('0000' + dColor.toString(16)).substr(-6)); $('#hexVal').html(' #' + ('0000' + dColor.toString(16)).substr(-6)+' '); juma.write("/peripherals/rgblight",('0000' + dColor.toString(16)).substr(-6)); });
